IMPLEMENTATION DETAILS
----------------------
1. TTL synchronizer:
   - 3-register chain: ttl_sync_1 → ttl_sync_2 → ttl_sync_3
   - ASYNC_REG attribute applied to ttl_sync_1 and ttl_sync_2
   - Rising-edge: ttl_edge = ttl_sync_2 AND NOT ttl_sync_3

2. USB / Timer edge detection:
   - Both already synchronous to system_clk — no synchronizer needed
   - usb_edge  = usb_trigger_in  AND NOT usb_prev
   - timer_edge = timer_trigger_in AND NOT timer_prev

3. Source selection gating:
   - "00" → ttl_active only
   - "01" → usb_active only
   - "10" → timer_active only
   - "11" → all active

4. Priority arbitration (combinational):
   - any_trigger = TTL OR (USB AND NOT TTL) OR (Timer AND NOT TTL AND NOT USB)
   - trigger_fired = any_trigger AND trigger_enable

5. Clean single-cycle output:
   - trigger_out registered from trigger_fired (1 cycle latency, width preserved)
   - trigger_source_id captured when trigger_fired='1'

6. 16-bit trigger counter:
   - Resets on reset='0' OR reset_counter='1'
   - Increments by 1 on trigger_fired='1'
